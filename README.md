# Decentralized Low-Rank Spatial Models

A Python implementation of decentralized algorithms for low-rank spatial data modeling and parameter estimation.

## Overview

This project implements decentralized estimation algorithms for spatial statistical models, focusing on low-rank approximations and distributed computation methods. The framework supports Decentralized Estimation (DE) approaches for spatial parameter estimation.


## Project Structure

```
├── src/                          # Source code
│   └── estimation_torch_real_data.py   # Core estimation algorithms using PyTorch
├── real_data/                    # Real data analysis
│  
├── experiments/                  # Experimental code
└── README.md                    # This file
```

## Installation

### Prerequisites

- Python 3.8+
- PyTorch
- NumPy
- Matplotlib
- Jupyter Notebook
- SciPy

### Setup

1. Clone the repository:
```bash
git clone <repository-url>
cd Dis_Spatial
```

2. Create and activate a virtual environment:
```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install torch numpy matplotlib scipy jupyter
```
